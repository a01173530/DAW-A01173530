MIME: Es la forma standard de mandar contenido a través de la red. Especifican tipos de datos, como por ejemplo texto, imagen, audio, etc. Se utiliza el sufijo correcto para este tipo de archivo.

¿Cuál es la diferencia entre Internet y la World Wide Web?


Internet: es la inmensa red de computadoras conectadas entre sí.


World Wide Web: la enorme colección de páginas sobre la red de computadoras.


¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

GET: Solicita una presentación de un recurso específico, solo deben recuperar datos del servidor.


HEAD: Solicita una respuesta idéntica a la de una petición GET pero sin el cuerpo de la respuesta.


POST: Se utiliza para enviar una entidad a un recurso específico, causa cambios en el estado o efecto en un servidor.


PUT: Reemplaza todas las representaciones actuales del recurso de destino con la carga útil de la petición.


PATCH: Es utilizado para aplicar modificaciones parciales a un recurso.


DELETE: Borra un recurso en específico.

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500? ¿Por qué?


Quizás si porque lo que este error nos dice es que hubo un problema con el servidor de la web y no especifica cual así que es trabajo del desarrollador solucionar el servidor aunque igual puede que sea culpa del navegante (lo cual es poco probable). ¿Cuáles son los principales controles de una forma HTML5?


form: Representa un formulario, consistiendo de controles que puede ser enviado a un servidor para procesamiento.


fieldset: Representa un conjunto de controles.


legend: Representa el título de un fieldset.


label: Representa el título de un control de formulario.

¿Cuál es el ciclo de vida de los sistemas de información?


Nacimiento: la necesidad o requerimiento por parte del usuario.


Desarrollo: se analizan los requerimientos y se elaborará un diseño que servirá como base del desarrollo.


Operación: El sistema ya está terminado y el usuario trabaja introduciendo datos y obteniendo información y reportes que soporten la operación de la empresa.


Mantenimiento: consiste en corregir los errores que se detectan en los programas o en las funciones que realice el sistema.


Muerte: cuando el sistema deja de ser necesario o cuando debe reemplazarse por otro mejor.

Datos obtenidos en:https://es.slideshare.net/MonicaNaranjo20/ciclo-de-vida-de-un-sistema-de-informacion-93192099#:~:text=Ciclo%20de%20vida%20de%20un%20sistema%20%EF%82%A7%20El%20ciclo%20de,del%20analista%20y%20del%20usuario.

¿Cuál es el ciclo de desarrollo de sistemas de información?


Planificación: Se prepara el diseño y posteriormente la implementación del sistema.


Análisis: tiene que ver con el análisis y se busca concretar los requisitos.


Diseño: Buscar cumplir los requisitos aplicables, identificar soluciones potenciales, evaluarlas y elegir la más conveniente.


Desarrollo: Inicio de la producción.


Integración y periodo de pruebas:Comprobar que el sistema cumpla los requisitos del negocio establecido.


Implementación: Instalación del hardware y software elegidos


Mantenimiento: Los ajustes requeridos del dia al dia.
