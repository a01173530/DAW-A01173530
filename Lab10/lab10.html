¿Por qué es una buena práctica separar el controlador de la vista? Porque evitamos tener código espagueti donde todos los elementos son dependientes. Y a la hora de que tengamos errores sería más fácil de identificar o hacer cambios en HTML.


Aparte de los arreglos $_POST y $_GET, ¿qué otros arreglos están predefinidos en php y cuál es su función?


$GLOBALS: Se refiere a todas las variables disponibles en el ámbito global. $_SERVER: Muestra información del entorno del servidor y de ejecución. $_FILES: Son variables de subida de ficheros HTTP. $_COOKIE: Son cookies HTTP. $_SESSION: Son variables de sesión. $_REQUEST: Para las variables HTTP Request. $_ENV: Para variables de entorno.

¿Qué es XSS y cómo se puede prevenir? Es un ciberataque que consiste buscar vulnerabilidades en una aplicación web para introducir un script dañino y atacar al propio sistema.


Algunas formas de prevenir este tipo de ciberataque son: Nunca incluir datos foráneos, solo desde lugares de confianza, codificar los datos a incluir en un elemento HTML, codificar y validar los atributos a incluir en un atributo HTML, codificar y validar el código JavaScript a incluir en un bloque de código JavaScript y HTML, codificar y validar el código CSS a incluir en los estilos HTML, codificar y validar las direcciones URL a incluir en un URL del HTML.
